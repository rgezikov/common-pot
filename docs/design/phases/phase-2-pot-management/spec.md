# Phase 2 — Pot Management

> Allow users to create and share a pot.

## Tasks

- Create a new pot (name, optional description)
- Auto-generate a secret link (UUID-based URL) on pot creation
- Access pot via secret link — no login required
- Add members to a pot (name only, no account needed)
- View pot details: name, member list
- Home page listing all pots (for the app owner/admin)
