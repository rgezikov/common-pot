# Phase 2 — Pot Management

> Allow users to create and share a pot.

## Tasks

- Telegram Login integration — authenticate users via Telegram Login Widget
- Create a new pot (name, optional description)
- Auto-generate an invite link for the pot
- Join a pot via invite link (authenticate with Telegram, get added as a member)
- View pot details: name, member list
- Home page showing all pots the logged-in user is a member of
