# Phase 6.3 — Pot Report

> Generate a printable summary of a pot's activity.

## Tasks

- Three-dot (⋮) menu on pot detail page with "Settings" and "Report" items
- Report accessible via the menu
- Report page is a standalone HTML page (no app chrome — no header/nav)
- Report includes:
  - Pot name, description, generation date
  - "Generated by Common Pot" attribution
  - Balance summary per member (Paid, Owed, Balance)
  - Settlement suggestions
  - Full list of drops (date, description, amount, paid by, split detail)
- Page is print-friendly (user can print to PDF via browser)
