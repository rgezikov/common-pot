# Phase 1003 — Shopping List: Simplified Add Form

> Reduce the add item form to a single input field on mobile.

## Change

Replace the separate name and note fields with a single input. The user can type just a name, or `name, note` to include a note in one step. The comma separates name from note; everything after the first comma becomes the note.

Placeholder text: `Item name (or name, note)`

Notes can also be added or edited via the item edit view.
