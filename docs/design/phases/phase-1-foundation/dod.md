# Phase 1 — Definition of Done

- [ ] Django project created and runs locally with `python manage.py runserver`
- [ ] Django app `pots` created within the project
- [ ] SQLite database configured and migrations run successfully
- [ ] Data models defined: `Pot`, `Member`, `Drop`, `Split`
- [ ] All models registered and visible in Django admin
- [ ] HTMX and TailwindCSS loaded via CDN in the base template
- [ ] Base template renders correctly in a mobile browser
- [ ] URL routing skeleton in place (placeholder views for main pages)
- [ ] Project runs on the server via gunicorn behind nginx
- [ ] Static files served correctly in production
