# Phase 18.2 — Menu Icon Consistency

> Use the hamburger icon (☰) for all dropdown menus across the app.

## Problem

The shopping list screen used ☰ (introduced in 18.1) while the home screen and pot detail screen still used ⋮ (vertical ellipsis). Inconsistent icons for the same UI pattern.

## Change

Replace ⋮ with ☰ on the home screen and pot detail screen.
