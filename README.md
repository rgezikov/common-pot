# Common Pot

Shared expense tracker for groups — trips, dinners, flat-shares.
Log who paid what, get settlement suggestions.
Accessible via browser and Telegram bot.

## Tech stack

- Python 3.12, Django 6, SQLite, gunicorn, nginx
- HTMX + TailwindCSS (CDN), no build step
- Telegram bot via python-telegram-bot 22
