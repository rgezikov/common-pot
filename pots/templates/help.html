What is Common Pot?

Common Pot tracks shared expenses for a group — trips, dinners, flat-shares. Everyone logs what they paid, and the app tells you who owes whom at the end.

Using the web app

  1. Log in with Telegram on the home screen.
  2. Create a pot or join one via an invite link.
  3. Add drops (expenses) with an amount, description, and who paid.
  4. Set split weights per member — leave all blank or put the same number to split equally.
  5. Check balances and settlement suggestions on the pot page.
  6. Share the invite link from Settings so others can join.

Using the Telegram bot

Add @common_pot_bot to a group chat, then:

  • /pot new — create a new pot for this group
  • /pot <invite_link> — link an existing pot (invite link from Settings)
  • /link — get the web app links (Open and Join)
  • /drop 45 dinner — log a drop paid by you
  • /drop 45 dinner @alice — log a drop paid by @alice
  • /drop =120-15 taxi — amount as a formula
  • /balance — show member balances
  • /settle — show settlement suggestions
  • /help — show this command list in chat

Drops added via the web app trigger a notification in the linked group chat.

Tips

  • On mobile, open pot links directly in Telegram — no login needed.
  • Use formulas like =200/3 or =120-15 in the amount field.
  • Leave all weights blank or put the same number to split equally.
  • Set a member's weight to 0 to exclude them from a specific drop.
